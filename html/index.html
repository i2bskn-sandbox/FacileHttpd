FacileHttpd

Sample http server on Linux/OSX.

Dependencies

  • Java 1.7
  • Gradle required for build.
  • daemon required for start as a daemon.

Start/Stop/Status

 $ cd /path/to/FacileHttpd
 $ ./bin/facile.sh {start|stop|status}
or 
 $ cd /path/to/FacileHttpd
 $ java -jar build/libs/FacileHttpd.jar conf/facilehttpd.xml

Configuration

<?xml version="1.0" encoding="UTF-8"?>

<facile>
  <servers>
    <server>
      <documentRoot>./html</documentRoot>
      <index>index.html</index>
      <port>9000</port>
    </server>
  </servers>
</facile>
